  • ms-clap.sass

Options

  • collapsed:
    Default: true
    Type: Boolean
    Values: true or false
    Description:
    Если true, то при открытии текущей панели, закрывать открытые (если такие есть).
    Есть возможность переопределения через data-атрибут data-clap-collapsed. Присваивать элементу, на котором вызывается плагин.
    Переопределить глобально можно с помощью конструкции $.fn.msclap.defaultOptions.collapsed = ...
    data-атрибут имеет приоритет выше, чем переопределение через javaScript
  • event:
    Default: 'click'
    Type: String
    Values: 'click' or 'mouseenter'
    Description:
    Событие, на которое происходить открытие/закрытие панели.
    Если 'click', то по клику.
    Если 'mouseenter', то при наведении.
  • someProperty:
    Default: someValue
    Type: String
    Values: otherValue
    Description:
    Описание

Глобальные настройки

Переопределять настройки плагина глобально можно через
$.fn.msclap.defaultOptions
Например:
$.fn.msclap.defaultOptions.someProperty = false;
