Документация

В плагине используются js-классы с префиксом tc. Сокращение от toggle-class.

Например, .tc__switcher-js,

Options

  • switchBtn:
    Defaults: null
    Type: String
    Example: '.some-class'
    Элемент, которым можно переключать класс. Использовать, когда нужно продублировать switcher, например, для мобилной версии
  • addBtn:
    Defaults: null
    Type: String
    Example: '.some-class'
    Элемент, которым можно только добавить класс
  • removeBtn:
    Defaults: null
    Type: String
    Example: '.some-class'
    Элемент, которым можно только удалить класс
  • other:
    Defaults: null
    Type: String, arr
    Example: '.some-class', or ['.some-class-1', '.some-class-2']
    Все остальные элементы, на которые будет добавлен класс
  • removeOutsideClick:
    Defaults: true
    Type: Boolean
    Удалить класс при клики вне этого элемента
    Примечания:
    Если значение этого параметра равно true, то при клике в ЛЮБОМ месте страницы все установленные ранее классы удаляются.
    В случае, если при клике на определенный элемент удаление не желательно, то на этот элемент необходимо добавить атрибут data-tc-stop (без значения).
  • cssScrollFixed:
    Defaults: false
    Type: Boolean or string (.class)
    Example: true, or 'some-class'
    Блокировка скролла после добавления класса
    Если указать значение true, то на html будет добавляться дополнительный класс css-scroll-fixed
    Также в значение параметра можно передать любой другой класс (!!!без точки).
  • modifiers: Defaults: 
    {
	init: 'tc--initialized',
	currentClass: 'tc--active'
}

Глобальные настройки

Поменять настройки плагина глобально можно через
$.fn.tClass.defaultOptions
Например:
$.fn.tClass.defaultOptions.removeOutsideClick = false;
