Документация

В плагине используются js-классы с префиксом tc__. Сокращение от toggle-class.

Например, .tc__switcher-js,

Options

  • switcher:
    Defaults: null
    Класс элемента, по которому происходит переключение класса
    Этот параметр необходимо указать, если свитчер добавляется динамически.
    Тогда плагин запускаем на статичной обертке, а в параметрах указываем switcher: (например) .tc__switcher-js
  • removeElements:
    Defaults: '.tc__close-js'
    Класс элементов, по которым происходит удаление класса
  • removeOutsideClick:
    Defaults: '.tc__outside-js'
    Удалить класс при клики вне этого элемента
    Примечания:
    1. Если значение этого параметра равно true, то при клике в ЛЮБОМ месте страницы все установленные ранее классы удаляются.
    Если же при клике на определенный элемент удаление не желательно, то на этот элемент необходимо добавить атрибут data-tc-stop (без значения).
  • modifiers: Defaults: 
    {
	init: 'tc--initialized',
	currentClass: 'tc--active',
	cssScrollFixed: 'css-scroll-fixed'
}

События

  • afterInit

    Вызов вконце функции afterInit()

    afterInit: function (e, el) {}

Глобальные настройки

Поменять настройки плагина глобально можно через

$.fn.tClass.defaultOptions

Например:

$.fn.tClass.defaultOptions.removeOnEsc = false;
