Простой попап

Если необходимо добавлять кнопки открытия динамически

Callback

Options

  • opener:
    Defaults: 
    '.ms-popup-d__opener-js'
    Класс элемента, по которому происходит открытие
  • popup:
    Defaults: 
    '.ms-popup-d__popup-js'
    Каласс попапа
  • closeBtn:
    Defaults: 
    '.ms-popup-d__close-js'
    Класс элемента, по которому происходит закрытие
  • outsideClose:
    Defaults: 
    true
    Закрыть все попапы при клике вне попапа
  • escapeClose: Defaults: 
    true
    Закрыть все попапы при клике на клавишу escape

События

  • afterInit

    Вызов вконце функции afterInit()

    afterInit: function (e, el) {}
  • afterOpened

    После открытия попапа

    afterOpened: function (e, el) {}
  • afterClosed

    После закрытия попапа

    afterClosed: function (e, el) {}

Глобальные настройки

Поменять настройки плагина глобально можно через

$.fn.simplePopupDynamic.defaultOptions

Например:

$.fn.simplePopupDynamic.defaultOptions.escapeClose = false;

Предупреждения !

  • Нельзя вызвать плагин больше, чем с одного класса на одном элементе. Для примера: откройте "Попап 1". Кликните по ссылке Открыть попап 2 - откроется попап "Попап 1". Кликните по ссылке Открыть попап 3 (новые классы) - ничего не произойдет. Хотя 
    <div class="ms-popup-d__init-js ms-popup-d__init-alt-js">
		<a href="#example-popup2" class="ms-popup-d__opener-js"><span>Открыть попап 2</span></a>
		<a href="#example-popup3" class="ms-popup-d__opener-alt-js"><span>Открыть попап 3 (новые классы)</span></a>
	</div>
    но ms-popup-d__init-alt-js не инициализируется, потому что инициализация проходит по первому вызову, а в скрипте первым вызывается ms-popup-d__init-js
