Options

  • switcher:
    Defaults: null
    Type: String
    Example: '.some-class'
    Элемент, которым можно ПЕРЕКЛЮЧАТЬ класс
  • adder:
    Defaults: null
    Type: String
    Example: '.some-class'
    Элемент, которым можно только ДОБАВЛЯТЬ класс
  • remover:
    Defaults: null
    Type: String
    Example: '.some-class'
    Элемент, которым можно только УДАЛЯТЬ класс
  • switchClassTo:
    Defaults: null
    Type: Object
    Example:
    Все остальные элементы, на которые будет добавлен класс
  • removeOutsideClick:
    Defaults: true
    Type: Boolean
    Удалить класс при клики вне этого элемента.
    Примечания:
    Если значение этого параметра равно true, то при клике в ЛЮБОМ месте страницы все установленные ранее классы удаляются.
    В случае, если при клике на определенный элемент удаление не желательно, то на этот элемент необходимо добавить класс stop-remove-class. Этот класс можно поменять в настройках плагина - modifiers.stopRemoveClass
  • removeEscClick:
    Defaults: true
    Type: Boolean
    Удалить класс при клики на клавишу Esc.
  • cssScrollFixed:
    Defaults: false
    Type: Boolean or string (.class)
    Example: true
    Блокировка скролла после добавления класса.
    Примечания:
    Если указать значение true, то на html будет добавляться дополнительный класс css-scroll-fixed
  • modifiers: Defaults:
    {
       initClass: null,
       activeClass: 'active',
       stopRemoveClass: 'stop-remove-class'
    }

Глобальные настройки

Поменять настройки плагина глобально можно через
$.fn.tClass.defaultOptions
Например:
$.fn.tClass.defaultOptions.removeOutsideClick = false;
