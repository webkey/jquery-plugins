API

Initialize

$('.element').nav(parameters);

For example:

$('.nav').nav({
  key: 'value'
});

Options

  • someProperty:
    Defaults: someValue
    Type: String
    Values: otherValue
    Description:
    Описание

Events

  • afterInit
    Description:
    Описание
    afterInit: function (e, nav) {}
  • afterHover
    Description:
    Вызов сразу после добавления класса hover:
    afterHover: function (e, nav, currentItem) {}
  • afterBlur
    Description:
    Вызов сразу после удаления класса hover:
    afterBlur: function (e, nav, currentItem) {}

Methods

  • afterInit
    Description:
    Описание
    afterInit: function (e, el) {}

Global settings

Переопределять настройки плагина глобально можно через
$.fn.nav.defaultOptions
Example:
$.fn.nav.defaultOptions.someProperty = value;
