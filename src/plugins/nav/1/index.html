API

Initialize

$('.element').nav(parameters);

For example:

$('.nav').nav({
  key: 'value'
});

Options

  • someProperty:
    Defaults: someValue
    Type: String
    Values: otherValue
    Description:
    Описание

  • afterInit
    Description:
    Описание
    afterInit: function (e, el) {}

Global settings

Переопределять настройки плагина глобально можно через
$.fn.nav.defaultOptions
Example:
$.fn.nav.defaultOptions.someProperty = value;
