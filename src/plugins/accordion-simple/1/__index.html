API

Initialize

$('.element').accordionSimple(parameters);

For example:

$('.Accordion-simple').accordionSimple({
  key: 'value'
});

Options

  • someProperty:
    Defaults: someValue
    Type: String
    Values: otherValue
    Description:
    Описание

Events

  • afterInit
    Description:
    Описание
    afterInit: function (e, el) {}

Methods

  • afterInit
    Description:
    Описание

Global settings

Переопределять настройки плагина глобально можно через
$.fn.accordionSimple.defaultOptions
Example:
$.fn.accordionSimple.defaultOptions.someProperty = value;
