Птицы

Включена возможность перестилить табы в селект с помощью Медиа развилок.
Чтобы увидеть перестроение табов в селект, нужно сузить браузер до 768 px.
$('.ms-tabs').msTabs({
  compactView: {
    elem: '.tabs__select-js',
    drop: '.tabs__select-drop-js',
    openClass: 'MY--SELECT-OPEN-CLASS'
  }
});

Документация

Files

  • __ms-tabs.html
  • jquery.tabs.js
  • example.js
  • ms-tabs.sass

Options

  • anchor:
    Defaults: '.tabs__anchor-js'
    Type: String
    Example: '.some-class'
    Переключатель таба
  • panels:
    Defaults: '.tabs__panels-js'
    Type: String
    Example: '.some-class'
    Обертка панелей
  • panel:
    Defaults: '.tabs__panel-js'
    Type: String
    Example: '.some-class'
    Панель таба
  • animationSpeed:
    Defaults: 80
    Type: Number
    Example: 1000
    Скорость анимации, в ms
  • collapsible:
    Defaults: false
    Type: Boolean
    Example: true
    Скрывать открытый таб по второму клику, или нет
  • modifiers: Defaults: 
    {
  initClass: null,
  collapsibleClass: null,
  activeClass: null
{

Глобальные настройки

Поменять настройки плагина глобально можно через
$.fn.tClass.defaultOptions
Например:
$.fn.tClass.defaultOptions.removeOutsideClick = false;
