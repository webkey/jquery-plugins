Документация

Used Files:

  • __ms-rolls.html
  • ms-rolls.sass
  • jquery.rolls.js
  • example.js

Options

  • item:
    Default: '.rolls__item-js'
    Type: String
    Values: '.your-class'
    Description:
    Общий ближайший родитель (Элемент) для переключателя и разворачивающейся панели (Далее Панель)
  • header:
    Default: '.rolls__header-js'
    Type: String
    Values: '.your-class'
    Description:
    Обертка для переключателя (Шапка)
  • panel:
    Default: '.rolls__panel-js'
    Type: String
    Values: '.your-class'
    Description:
    Панель. Элемент, который будет открываться/закрываться
  • hand:
    Default: '.rolls__hand-js'
    Type: String
    Values: '.your-class'
    Description:
    Переключатель. Элемент по клику (ховеру) происходить открытие или закрытие Панели
  • event:
    Default: 'click'
    Type: String
    Values: 'click' or 'mouseenter'
    Description:
    Событие, на которое происходить открытие/закрытие панели.
    Если 'click', то по клику.
    Если 'mouseenter', то при наведении.
  • animationSpeed:
    Default: 300
    Type: Number
    Values: Время в милисекундах
    Description:
    Скорость анимации Панели
  • collapsed:
    Default: true
    Type: Boolean
    Values: true or false
    Description:
    Если true, то при открытии текущей панели, закрывать открытые (если такие есть).
    Есть возможность переопределения через data-атрибут data-rolls-collapsed. Присваивать элементу, на котором вызывается плагин.
    Переопределить глобально можно с помощью конструкции $.fn.msRolls.defaultOptions.collapsed = ...
    data-атрибут имеет приоритет выше, чем переопределение через javaScript
  • accessibility:
    Default: false
    Type: Boolean
    Values: true/false
    Description:
    Опримизирует переключение фокуса по табу, но при этом снижает скорость выполнения скрипта
  • modifiers:
    Default: modifiers: { activeClass: 'rolls-active' }
    Type: Object
    Values: Some class
    Description:
    Модификаторы (классы) состояний

События

  • afterInit
    afterInit: function (e, el) {}
    $('.ms-rolls-js').on('msRolls.afterInit', function () {
  console.log('Plugin MS-Rolls has initialized!');
}).msRolls({});
  • beforeOpen
    Description:
    Вызывается перед началом открытия текущей Панели
    beforeOpen: function (e, el) {}
  • afterOpen
    Description:
    Вызывается после окончания открытия текущей Панели
    afterOpen: function (e, el) {}
  • beforeClose
    Description:
    Вызывается перед началом закрытия текущей Панели
    beforeClose: function (e, el) {}
  • afterClose
    Description:
    Вызывается после окончания закрытия текущей Панели
    afterClose: function (e, el) {}

Пример исспользования событий

  • $('.rolls-js').msRolls({
  beforeOpen: function (e, el) {
    console.log('.beforeOpen');
  }
})

Методы

var myRolls = $('.rolls-js').msRolls();
myRolls.msRolls('название метода', 'целевая Панель' [, callback function]);
  • open
    Description:
    Открывает указанную панель
  • close
    Description:
    Закрывает указанную панель

Пример исспользования методов

  • Открыть по клику на .example-btn Панель с id #panelId
    var myRolls = $('.rolls-js').msRolls();

$('.example-btn').on('click', function () {
  myRolls.msRolls('open', $('#panelId'), function () {
    console.log('Панель #hashExpl открыта!');
  });
});

Глобальные настройки

Переопределять настройки плагина глобально можно через
$.fn.msRolls.defaultOptions
Например:
$.fn.msRolls.defaultOptions.animationSpeed = 1000;
