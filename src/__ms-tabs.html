Документация

Files

  • __ms-tabs.html
  • jquery.ms-tabs.js
  • common-ms-tabs.js
  • ms-tabs.sass

Options

  • anchor:
    Defaults: '.tabs__anchor-js'
    Type: String
    Example: '.some-class'
    Переключатель таба
  • panels:
    Defaults: '.tabs__panels-js'
    Type: String
    Example: '.some-class'
    Обертка панелей
  • panel:
    Defaults: '.tabs__panel-js'
    Type: String
    Example: '.some-class'
    Панель таба
  • animationSpeed:
    Defaults: 80
    Type: Number
    Example: 1000
    Скорость анимации, в ms
  • collapsed:
    Defaults: false
    Type: Boolean
    Example: true
    Скрывать открытый таб по второму клику, или нет
  • modifiers: Defaults: 
    {
  init: 'tabs--initialized',
  activeClass: 'tabs-active'
{

Глобальные настройки

Поменять настройки плагина глобально можно через
$.fn.tClass.defaultOptions
Например:
$.fn.tClass.defaultOptions.removeOutsideClick = false;
